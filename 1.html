      4. HyperText Markup Language (HTML) (HTML, Cultural Language: Herbal Text Markup Language, HyperText Marku Language) is thedominant markup language for web pages

      HTML stands for Hyper Text Markup Language and is a Markup Language for writing world wide web documents. HTML is composed of several tags, and each tag can be used to create a document in a desired format. Internet documents use the principle of Hyper Text to link multiple documents so that various information can be easily searched and viewed. With such a variety of expression possibilities and ease of use, it has become a driving force in popularizing the World Wide Web in a short time.
